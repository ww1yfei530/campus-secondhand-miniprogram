# 同校闲集 · 校园二手闲置交易小程序
一款面向高校学生的同校实名认证二手闲置交易小程序，主打「同校信任+线下履约」，解决校园二手交易信息杂乱、信任成本高、履约无保障的痛点。

## 🎯 项目定位
- **目标用户**：在校大学生
- **核心价值**：基于校园实名认证的信任背书，提升二手闲置流转效率
- **MVP目标**：验证「同校学生愿意使用专属平台完成二手交易」的核心假设

## ✨ 核心成果
- 冷启动覆盖本地3所高校，累计访问用户XXX人，校园认证转化率XX%
- 累计发布闲置商品XXX件，用户咨询XXX次，成功交易XX单，订单转化率XX%
- 独立完成从用户调研、需求分析、产品设计到技术落地的全流程

## 📝 产品工作内容
1. **需求洞察**：通过用户访谈+问卷调研，梳理买卖双边核心痛点，明确「同校信任」为核心差异化
2. **MVP规划**：用MoSCoW法则筛选需求，砍掉非核心功能，6周完成从0到1上线
3. **功能设计**：完成信息架构设计、核心用户流程梳理、全页面原型设计
4. **PRD撰写**：输出商品发布、浏览搜索、私信沟通、订单履约等核心功能PRD
5. **数据复盘**：上线后跟踪核心指标，基于用户反馈输出V1.1迭代规划

## 🛠️ 技术实现
- 前端：微信小程序原生开发（WXML/WXSS/JS）
- 后端：微信云开发（云数据库 + 云函数 + 云存储）
- 核心功能：校园认证、商品发布、分类搜索、私信沟通、订单履约
