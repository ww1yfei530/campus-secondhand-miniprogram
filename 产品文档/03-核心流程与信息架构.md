# 核心用户流程与信息架构
## 买家核心交易流程
打开小程序 → 浏览首页/分类筛选/搜索商品 → 进入商品详情 → 发起私信沟通 → 确认商品信息 → 下单（选择交付方式、填写时间地点）→ 支付 → 线下收货 → 确认收货 → 交易完成

## 卖家核心发布流程
打开小程序 → 点击发布按钮 → 实名认证校验 → 填写商品信息（图片、标题、分类、成色、价格、描述）→ 提交发布 → 商品上架展示 → 接收买家咨询 → 买家下单 → 标记已交付 → 买家确认收货 → 交易完成

## 信息架构
底部3个Tab：首页、消息、我的
- 首页：搜索框、分类Tab、商品信息流、发布悬浮按钮
- 消息：会话列表、聊天页（顶部绑定商品卡片）
- 我的：个人信息、认证状态、我发布的、我买到的、我卖出的、我的收藏
