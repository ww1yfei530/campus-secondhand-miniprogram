# 商品发布功能 PRD
## 功能概述
让已认证的学生用户快速发布闲置商品，是供给侧的核心入口。

## 入口路径
首页右下角悬浮+号按钮、个人中心「我要发布」

## 字段规则
| 字段 | 必填 | 校验规则 |
|---|---|---|
| 商品图片 | 否 | 最多9张，支持jpg/png/webp，单张不超过5MB |
| 商品标题 | 是 | 4-30字 |
| 商品分类 | 是 | 5个固定选项：教材、数码电子、生活用品、宿舍家具、其他 |
| 商品成色 | 是 | 5个固定选项：全新、几乎全新、轻微使用、明显使用、破损 |
| 商品价格 | 是 | 数字，范围0.01-9999元，保留两位小数 |
| 商品描述 | 否 | 最多500字 |

## 交互规则
1. 失焦实时校验，错误提示在输入框下方，不弹窗打断用户
2. 点击发布后按钮置灰，防止重复提交
3. 无草稿箱，离开页面弹窗提示「离开后内容将不保存」

## 异常流程
1. 未认证用户点击发布：弹窗引导去认证，跳转认证页
2. 图片上传失败：提示重试，已上传的图片保留
3. 网络错误提交失败：保留所有已填内容，提示重试
